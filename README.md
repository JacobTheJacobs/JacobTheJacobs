## What I've built

- **[Antigravity CLI Copilot](https://marketplace.visualstudio.com/items?itemName=jacobthejacobs.antigravity-chat)** — Google's Antigravity CLI inside VS Code: streaming replies, live tool rows, generated images inline, and follow-ups queued instead of cancelling the turn.
- **[Grok CLI Copilot](https://marketplace.visualstudio.com/items?itemName=jacobthejacobs.grok-cli-copilot)** — The Grok CLI inside VS Code — chat, agentic edits, and full file context in a focused sidebar.
- **[Claw Mate](https://apps.microsoft.com/detail/9NJH9X9KDG5P)** — Desktop AI pet that watches your coding agents (Claude Code, Codex, OpenClaw) and surfaces approvals the moment they're needed. · [Studio](https://claw-mate-studio.web.app/)
- **[TradingAgents Control Room](https://github.com/JacobTheJacobs/TradingAgents-Control-Room)** — Multi-agent trading floor: AI analysts debate a stock live in pixel-art — analysts → bull/bear debate → trader → risk → decision. React + Phaser + FastAPI, data from the Fin-Node CDN.
- **[Llama Menu](https://github.com/JacobTheJacobs/llamacpp-menubar)** — macOS menu bar control for local llama.cpp — runs GGUF models at the largest context your Mac can actually hold, sized from real GGUF metadata.
- **[Hermes Menu](https://github.com/JacobTheJacobs/hermes-menubar)** — macOS menu bar control for the Hermes gateway — start, stop and watch it, with status read straight from launchd rather than scraped from CLI output.
- **[Corporate Brawl](https://corporate-brawl.web.app)** — A chaotic office-brawl browser game, built in Three.js.
- **[Floor Planner](https://floorplan-29233.web.app/)** — Draw rooms in 2D and watch them rise in 3D — live, in the browser.
- **[PixelBoy](https://pixel-boy-studio.web.app/)** — Retro pixel sprite editor: frame-by-frame animation, a 60+ color palette, and PNG spritesheet export.
- **Browser SDK** — Private browser-automation SDK that drives real Chrome with a vision model and a step planner. 🚧
- **PhantomPilot** — Autonomous Android agent that reads the screen and taps through real apps, hands-free. 🚧
- **[Alexa NFT Trending](https://github.com/JacobTheJacobs/AlexaNftTrending)** — Alexa skill that reads out the top trending NFTs by voice (.NET/C#).
